## Touch Gestures

### Tap

```
Duration: < 300ms
Visual: Ripple effect
Audio: Optional click
Haptic: Light (10ms)
```

### Long Press

```
Duration: 500-800ms
Visual: Scale down (0.97)
Haptic: Medium (20ms)
Use: Context menu, selection mode
```

### Swipe

```
Threshold: > 50px in direction
Velocity: > 0.3 px/ms for quick action

Left swipe: Delete, archive
Right swipe: Complete, favorite
```
